# Stock Trading Platform

## Overview

A Java-based Stock Trading Platform that simulates a basic stock market environment. Users can view market data, buy and sell stocks, and track portfolio performance.

## Features

* Market data display
* Buy stock functionality
* Sell stock functionality
* Portfolio tracking
* Balance management
* Object-Oriented Programming concepts
* Real-time transaction simulation

## Technologies Used

* Java
* OOP (Classes and Objects)
* ArrayList
* HashMap
* Scanner Class

## Classes

* Stock.java
* User.java
* Transaction.java
* StockTradingPlatform.java

## How to Run

Compile:

javac Stock.java
javac User.java
javac Transaction.java
javac StockTradingPlatform.java

Run:

java StockTradingPlatform
